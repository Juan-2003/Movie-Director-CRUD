# Movie-Director-CRUD

Este proyecto es un **CRUD completo** desarrollado con las siguientes tecnologías:  

- **Backend:** C# con .NET  
- **Frontend:** React
Se pueden realizar las operaciones de GET, PUT, POST y DELETE para las entidades Director y Movie
